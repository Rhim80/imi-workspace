# Create Command

커스텀 명령어를 만듭니다.

## 실행

```bash
/create-command
```

## 대화형 생성

Claude Code가 질문하며 명령어를 만들어줍니다:

1. "명령어 이름은?" → 예: `meeting-prep`
2. "설명은?" → 예: "미팅 준비 자료 생성"
3. "어떤 작업을 수행하나요?" → 단계별 설명

## 활용 아이디어

- `/weekly-plan` - 주간 계획 생성
- `/expense` - 경비 정리
- `/client-brief` - 클라이언트 브리핑
- `/lecture-outline` - 강의 개요 작성

## 출력

`.claude/commands/명령어이름.md` 파일 생성
