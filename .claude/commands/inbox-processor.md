# Inbox Processor

Inbox를 정리합니다.

## 실행

```bash
/inbox-processor
```

## 수행 작업

1. `00-inbox/` 폴더의 모든 파일 확인
2. 각 파일 분석:
   - 프로젝트 관련 → `10-projects/`
   - 운영 업무 → `20-operations/`
   - 학습 자료 → `30-knowledge/`
   - 개인 메모 → `40-personal/`
   - 참고 자료 → `50-resources/`
3. 이동 제안 및 실행

## Inbox Zero 원칙

- Inbox는 임시 공간
- 주 1회 정리
- 빠른 캡처 > 완벽한 분류
