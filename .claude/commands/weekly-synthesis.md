# Weekly Synthesis

주간 회고를 작성합니다.

## 실행

```bash
/weekly-synthesis
```

## 수행 작업

1. `40-personal/42-weekly/` 폴더 확인
2. 이번 주차 파일(`YYYY-WXX.md`) 생성
3. 이번 주 Daily Notes 분석
4. 주간 회고 작성 도움

## 템플릿

`00-system/01-templates/weekly-review-template.md`

## Weekly Review 내용

- 이번 주 주요 성과
- 배운 점
- 개선할 점
- 다음 주 계획
